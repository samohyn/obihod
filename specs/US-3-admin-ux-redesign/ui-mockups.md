## 0. Ключевой визуальный принцип

> **«Штамп бригадира на крафтовой бумаге».** Каждое решение — плотная типографика без засечек, кремовый фон, зелёно-янтарный штамп, без градиентов и теней-glow. Сверху UX-принципа `ux` «никакого экрана без подсказки» накладывается визуальный: **статус и действие — всегда в одном кадре.** Никаких «где тут кнопка», никаких «а что это за бейдж» — цвет = смысл, форма = приоритет, размер = частота.

Производные:
- **Primary — зелёный Обихода `#2d5a3d`.** Одна доминирующая кнопка на экран.
- **Янтарь `#e6a23c` — только сигнал.** Preview-баннер, publish-gate error, статус «черновик». Не декор.
- **Плотность UI > воздух.** Это инструмент, не витрина. Строка таблицы — 44 px (WCAG touch), не 64 px.
- **Цифры везде `tabular-nums`.** ID заявок, цены, счётчики выравниваются колонкой — глаз ищет число, а не текст.

| Токен | px | Применение |
|---|---:|---|

### 1.4 Радиусы

| Токен | px | Применение |
|---|---:|---|
| `--r-1` | 4 | Checkbox, status-dot, bulk-select, inline-badge |
| `--r-2` | 8 | Input, dropdown item, tag, button |
| `--r-3` | 12 | Card (DashboardTile, BlockPaletteCard, InlineHelp) |
| `--r-4` | 16 | Modal, sticky-bar (snap top/bottom — только два верхних/нижних угла) |

### 1.5 Тени

Плоский стиль — **без blur-glow**. Только edge-shadow для «подъёма» карточки.

| Токен | CSS | Применение |
|---|---|---|
| `--sh-none` | `none` | Default card state |
| `--sh-1` | `0 1px 2px rgba(28, 28, 28, 0.05)` | Hover row, tab-active |
| `--sh-2` | `0 2px 6px rgba(28, 28, 28, 0.08)` | Hover DashboardTile/BlockPaletteCard |
| `--sh-3` | `0 4px 12px rgba(28, 28, 28, 0.10)` | Dropdown, tooltip |
| `--sh-4` | `0 8px 24px rgba(28, 28, 28, 0.14)` | Modal overlay, AddBlockPalette dialog |
| `--sh-focus` | `0 0 0 2px #2d5a3d, 0 0 0 4px rgba(45, 90, 61, 0.20)` | :focus-visible на любом интерактиве |

### 1.6 Анимации

Минимум motion. Micro-interactions — функциональные, не декоративные.

| Токен | Duration | Easing | Применение |
|---|---|---|---|
| `--t-fast` | 100ms | `ease-out` | Hover background, status-dot fade |
| `--t-base` | 150ms | `cubic-bezier(0.2, 0, 0, 1)` | Button hover, tile translateY, focus-ring |
| `--t-medium` | 200ms | `cubic-bezier(0.2, 0, 0, 1)` | Dropdown open, tooltip fade |
| `--t-slow` | 300ms | `cubic-bezier(0.2, 0, 0, 1)` | Modal fade-in, publish-gate banner collapse |
| `--t-drag` | 150ms | `cubic-bezier(0.2, 0, 0, 1)` | Block reorder смещение соседей |

**Reduced-motion (`prefers-reduced-motion: reduce`):** все duration → 0.01ms, transform → opacity.

Ключевой паттерн — **hover lift на карточке tile**:
```css
transform: translateY(0);
transition: transform var(--t-base), box-shadow var(--t-base), border-color var(--t-base);
/* hover */ transform: translateY(-2px); box-shadow: var(--sh-2); border-color: var(--primary);
```

### 1.7 Брейкпоинты

**Desktop only** (BA ограничение):
- `viewport-min: 1280 px` — baseline.
- `1440 px` — комфортный, сетка tile 3×2 без скролла.
- `1920+ px` — max-content-width 1600 px с центрированием (sidebar 240 px + main 1360 px).

Mobile/tablet не поддерживается. При `< 1280 px` — показываем полноэкранную плашку: «Админка Обихода работает с экрана от 1280 px. Откройте на ноутбуке или подключите монитор». Это часть `BrandHeader` компонента (§3.8).

- **Loaded normal:** см. основной макет.
- **Error (API не поднялся):** Greeting — минимальный. Widget → `StateError` (§4) «Не удалось загрузить. [Повторить]». Tile остаются — они статичны.

#### Общие edge-cases

1. **Очень длинный title карточки** (DashboardTile, BlockPaletteCard): `text-overflow: ellipsis`, `line-clamp: 2`, tooltip на hover с полным текстом.

3. **Очень длинный breadcrumb:** средние крошки сворачиваются в «...», tooltip на hover.
4. **User offline (no network):** баннер сверху «Нет соединения с сервером. [Повторить]» — bg=#b54828 text=#f7f5f0.
5. **Token expired:** `draftMode` stale — PreviewBanner меняется на «Сессия preview истекла. [Обновить]».

## Часть 6. Accessibility чек-лист (WCAG 2.2 AA)

1. **Контраст ≥ 4.5:1** на всех body-text combinations — проверено в art-concept §8. `--muted #6b6256` на `--bg #f7f5f0` = 5.2:1 (baseline).
2. **Контраст ≥ 3:1 UI** — янтарь #c18724 на #f7f5f0 = 3.3:1 (только UI, не текст).
3. **Focus-visible ring** — `--sh-focus` 2 px solid #2d5a3d + glow 4 px transparent. Present on all buttons, links, inputs, tiles, cards, drag-handles, tabs, checkbox.
4. **Touch-target ≥ 44 px** — все интерактивы (buttons, tiles, row-actions). Drag-handle 24×56 = 56 высоты ≥ 44.
5. **Keyboard-only drag-drop** — см. §2.3 блоков.
6. **Aria-label на навигационных tile** — «Добавить район — 3 шага, около 5 минут».
7. **Landmark roles** — `<header>`, `<nav>`, `<main>`, `<aside>`, `<article>`.
8. **Heading hierarchy** — H1 → H2 → H3 без пропусков на каждой странице.
9. **Screen-reader announcements** — `aria-live="polite"` для autosave, `aria-live="assertive"` для PublishGateBanner, drag-drop, preview-status.
10. **Prefers-reduced-motion** — все transitions → 0.01 ms, transform → opacity где возможно.
