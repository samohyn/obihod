## Резюме запроса

Финальный SEO-аудит: убедиться, что сайт (а) технически соответствует стандартам Яндекса и Google, (б) реально виден в индексе, (в) цитируется нейро-поиском.

Блоки:

1. **Технический SEO-аудит.**
   - `robots.txt` корректный, sitemap.xml зарегистрирован в Я.Вебмастер + GSC, проиндексирован.
   - Canonical на всех страницах, нет дублей.
   - `hreflang` (если добавим en для B2B — сейчас не планируется).
   - Schema.org: Organization, LocalBusiness, Service, FAQPage, BreadcrumbList, Article, Review — валидируются в Rich Results Test + Я.Валидатор.
   - OpenGraph + Twitter Card на всех страницах.
   - Core Web Vitals: все страницы в «зелёной зоне» Я.Метрики и PSI.
   - SSL, HTTP → HTTPS, WWW → non-WWW 301.
   - Structured pagination, `rel=canonical`.
2. **Индексация.**
   - Я.Вебмастер: все нужные страницы в поиске, нет «исключено».
   - GSC: Coverage report чистый.
   - IndexNow подключён (пушим URL на изменения).
   - Проверка на cloaking / скрытый текст / анти-паттерны.
3. **Позиции.**
   - TopVisor / Rush Analytics / собственная проверка — замер позиций по 100–300 ключей из US-4.
   - Baseline для дальнейшего роста.
4. **Нейро-SEO.**
   - Тест 100–200 запросов через Я.Нейро / GigaChat / Perplexity / Alice Pro.
   - Замер цитируемости Обихода в ответах.
   - Рекомендации по улучшению: TL;DR-блоки, FAQ-схема, E-E-A-T, lastReviewed, reviewedBy, цитируемые числа/факты.
5. **Поведенческие факторы.**
   - Я.Метрика: время на странице, глубина, отказы, скролл-депс, клик-map, Вебвизор (sampling).
   - Cohort-анализ по каналам (Директ, SEO, прямой, рефереры).
   - Рекомендации на улучшение: заглушки-pops, внутренние ссылки, sticky-CTA, A/B-гипотезы (bake в US-8 follow-up).
6. **Локальное SEO.**
   - Я.Бизнес / 2ГИС: карточки заведены, адреса верны, фото загружены.
   - Геосигналы (geo-тегирование фото, Я.Карты widget на programmatic).
   - Отзывы: Я.Карты, 2ГИС, Отзовик — мониторинг.

Выход US-10: **SEO Launch Audit Report** с цветными вердиктами (🟢/🟡/🔴 — **в отчёте, не в интерфейсе**) по каждому пункту + backlog-тикеты на 🔴/🟡.

## Открытые вопросы к оператору

1. **Я.Бизнес / 2ГИС** — карточки уже созданы или делаем в US-10?
2. **IndexNow** — подключаем ко всем CMS-хукам (afterChange → push URL)? Есть ADR от `tamd`?
3. **TopVisor/Rush Analytics** — что подключаем для замера позиций? ~1 500–3 000 ₽/мес.
4. **Что считаем «success» для запуска программы?** Рекомендация: индексация 100% нужных страниц + CWV зелёные + ≥ 30% кластеров в топ-10 Яндекса МО через 3 месяца. Согласен на такие KPI?

## Предварительный бриф для BA

1. **Цель:** сайт технически образцовый и реально виден в поисковиках + нейро.
2. **Метрики:**
   - 100% нужных страниц в индексе Яндекса и Google.
   - 100% критических schema-проверок зелёные.
   - CWV зелёные на 95%+ URL.
   - Нейро-цитируемость ≥ 30% по тест-сету.
3. **Границы:**
   - IN: аудит, отчёт, backlog-тикеты, настройка Я.Бизнес/2ГИС, IndexNow.
   - OUT: исправления найденных проблем (создаются follow-up US).
4. **Состав команды:**
   - `seo2` (ведущий) — аудит и отчёт
   - `seo1` — ревью KPI и рекомендаций
   - `re` — нейро-SEO тест
   - `pa` — поведенческий анализ Я.Метрики

   - `out` — accept + финальный релиз программы
