## Резюме запроса

Собрать промышленное семантическое ядро под 4 услуги (арбористика, чистка крыш, вывоз мусора, демонтаж) × Москва/МО с пилотной географией (Одинцово, Красногорск, Мытищи, Химки, Истра, Пушкино, Раменское, Жуковский), провести конкурентный SEO-аудит **liwood.ru** и других конкурентов из [contex/01](../../../contex/01_competitor_research.md), [contex/04](../../../contex/04_competitor_tech_stacks.md), и на выходе дать:

1. Кластеризованное ядро (pillar / sub-services / programmatic service×district / B2B-кластер / блог) с частотностями, сложностью, интентом (коммерческий/локальный/информационный/нейро).
2. Решение по **глубине дробления услуг** (сколько sub-service страниц нужно, чтобы догнать liwood.ru, кто и как дробится глубже 22 известных).
3. Список ключей для нейро-SEO (Яндекс.Алиса, GigaChat, Perplexity, Нейро-ответы) и требования к контенту под них.
4. Temник на контент (pillar-страницы, programmatic, блог, FAQ, кейсы) — вход для US-5 (сайтмап) и US-6 (наполнение).

## Сервисы

**Решение оператора (2026-04-25, финал):** Keys.so + Just-Magic уже есть, **+ подключаем Topvisor** (~1 500–3 000 ₽/мес) вместо Key Collector. Topvisor закрывает US-4 (парсинг подсказок + Wordstat + кластеризация) **и** US-10 (мониторинг позиций) одним сервисом, есть REST API → `re` автоматизирует без участия оператора. Ahrefs не берём. Serpstat — открытый вопрос для US-7 (см. ниже).

### Стек US-4

| Сервис | Назначение | Что закрывает | Действие |
|---|---|---|---|
| **Keys.so** ✅ есть | SERP-разведка по Яндексу, выгрузка ключей конкурентов, видимость, пересечения доменов | Парсинг ключей 15 конкурентов + базовая частотность (~200 млн запросов) | передать доступ |
| **Just-Magic** ✅ есть | Hard-кластеризация по ТОП-10 Яндекса (≥ 30% URL пересечение → 1 страница) | Группировка 2 000–3 000 ключей в кластеры | передать доступ |
| **Topvisor** 🆕 покупаем | Парсинг Yandex/Google Suggest + точная частотность `"!запрос"` / `"[!запрос]"` по МО + сезонность Wordstat + REST API + **мониторинг позиций** (US-10) | +30–40% long-tail + точные частоты + автоматизация через `re` | зарегистрировать аккаунт + API-ключ ~1 500–3 000 ₽/мес |
| **Yandex.Wordstat (web)** | Дополнительная сверка частот | Бесплатно | — |

**Итого US-4: ~1 500–3 000 ₽/мес** (Topvisor; точная цена зависит от объёма ключей и частоты съёма позиций).

**Почему Topvisor вместо Key Collector 4:**
- KK4 — десктоп под Windows, нет API, нужно работать в интерфейсе руками → оператор в петле
- Topvisor — облако с REST API → `re` автоматизирует целиком, оператор вне петли
- Topvisor покрывает 3 эпика (US-4, US-7 частично, US-10) одним сервисом, KK4 — только US-4
- РФ-карта работает в обоих, цена сопоставима за период проекта

### Подключаем на US-7 (content + SEO audit)

| Сервис | Назначение | Цена | Оплата из РФ |
|---|---|---|---|
| **Serpstat Lite** | Backlinks конкурентов + content gap + Яндекс-видимость в одном | $69/мес ≈ ~7 000 ₽/мес | ✅ карты РФ работают |

### Не подключаем

| Сервис | Почему пропускаем |
|---|---|
| **Ahrefs** | $129/мес ≈ 13 000 ₽/мес, нужна международная карта. Слаб в Яндексе и Wordstat. Serpstat покрывает те же задачи дешевле и проще |
| **PixelTools / PR-CY Local** | Дублируется с Keys.so + Wordstat |
| **TopVisor / Rush Analytics (мониторинг)** | Мониторинг позиций — задача US-10, не US-4 |

## Предварительный бриф для BA

1. **Бизнес-цель:** понять, какие запросы в МО приносят клиентов конкурентам (и особенно liwood.ru), собрать ядро, дать основу для US-5 (полная IA сайта) и US-6 (наполнение).
2. **Метрика успеха:**
   - Ядро ≥ 1 500 кластеризованных ключей с частотностями по МО.
   - Конкурентный отчёт по liwood.ru: топ-50 самых трафиковых страниц, их URL-паттерны, длина контента, глубина каталога, backlink-профиль (если Ahrefs/Serpstat).
   - Контентный план с 100% покрытием pillar + sub-services + programmatic (4 × 15 районов минимум) + нейро-запросы + блог (30–50 тем).
3. **Границы задачи:**
   - IN: парсинг Wordstat/Keys.so, кластеризация, SERP-анализ liwood.ru, темник, нейро-SEO-карта.
   - OUT: написание контента (это US-6), имплементация в Payload (это US-6 и US-3), настройка позиций и мониторинга (US-7/US-10).
4. **Риски:**
   - Wordstat лимиты и антибот — понадобится прокси или XML-лимит (входит в стоимость).
   - liwood.ru может иметь спец-соглашения с Ahrefs (cloaking/noindex) — данные неточны, нужен ручной парсинг.
5. **Состав команды:**
   - `ba` — scope + AC
   - `seo1` (ведущий) — архитектура ядра, SERP-стратегия, нейро-SEO требования
   - `re` (research-analyst) — парсинг liwood.ru, SERP-TOP Яндекс/Google, конкурентный отчёт, SEO-tools integration
   - `cw` — валидация темника на соответствие TOV
   - `seo2` — консультирует по техничке (чтобы ядро учитывало возможности schema.org / FAQPage / HowTo)
   - `qa1` — qa-checklist: каждая pillar-тема имеет ≥ 3 программатик-стендов и ≥ 1 FAQ-кластер
   - `out` — accept
