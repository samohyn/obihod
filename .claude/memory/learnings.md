- 2026-04-23 · admin · `site/app/(payload)/admin/importMap.js` коммитится в репо и устаревает при добавлении global/collection/custom-component. - 2026-04-23 · deploy · Next.js 16 ISR кеширует prerender (`s-maxage=3600`, `stale-while-revalidate=31532400`) — после PM2 restart первый запрос может отдать старую статику из `.next` до инвалидации. Force через `/api/revalidate?tag=<X>&url=<path>` с `REVALIDATE_SECRET` — единственный надёжный способ сбросить без ожидания · **Why:** uptime=2s не гарантирует свежий HTML; проверять по content, не по uptime
