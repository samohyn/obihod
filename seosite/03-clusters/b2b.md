# B2B — УК / ТСЖ / FM / застройщики / госзаказ

**Pillar:** `b2b`
**Wave 2 версия:** v0.5 (передаётся seo1 для финализации).
**Кластеризация:**
- 50 ключей — Just-Magic `grp_onl` (Yandex TOP-10, ya_lr=213, ya_lrws=1, threshold grp3).
- ~1550 ключей — локальная морфо-кластеризация (`local_cluster.py`),
  потому что Just-Magic API упорно возвращал 503 на тяжёлые POST в окне сессии.
- Полная кластеризация: запустить `seosite/scripts/jm_cluster_batched.py`
  когда JM-сервис восстановится (`info`-ping → 200 → submit мелкими батчами).
**Частоты wsfreq:**
- ~200 ключей с реальными wsfreq (Just-Magic `wsfreq`, Москва+МО).
- Остальные 1400+ — `freq=0` означает «не запрошено», не «нет частоты».
  Дозабор: `seosite/scripts/jm_wsfreq_micro.py` (50-kw батчи, ~1₽/батч).

**Полная таблица:** [seosite/02-keywords/normalized.csv](../02-keywords/normalized.csv).
