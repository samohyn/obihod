### 2.2 · arboristika

| Поле | Значение |
|---|---|
| **ОКВЭД** | 02.40 (услуги в области лесоводства) + 81.30 (благоустройство ландшафта) |
| **ГОСТ** | ГОСТ Р 56198-2014 (озеленение городских территорий — требования к работам) · ГОСТ Р 53053-2008 (порубочные остатки — переработка) |
| **СНиП / СП** | СНиП 12-04-2002 (безопасность работ на высоте) — sustained до Минтруда №782н |
| **ФЗ** | Лесной кодекс РФ (если работы вне населённого пункта) · 89-ФЗ (для порубочных остатков как отходов IV класса) |
| **Местная регуляторика** | Постановление Правительства Москвы № 743-ПП «О защите зелёных насаждений» — порубочный билет обязателен для деревьев Ø свыше 8 см на муниципальной земле (НЕ на частной СНТ-территории) |
| **КоАП** | ст. 8.28 КоАП («Незаконная рубка лесных насаждений» — штраф юр.лицу 250-500 тыс. ₽) — для случая работ без порубочного билета на муниципальной земле |
| **Допуски** | Минтруда №782н 3-я группа по высоте (sustained `credentials.md` §3.2) |
| **СРО** | не требуется для базового спила; обязательна для работ в зонах действующих коммуникаций |

**Тех. процесс:**

1. Заявка → фото дерева + участок (частный / СНТ / муниципальная)
2. Если муниципальная земля → проверка порубочного билета (если нет — оформление от УК или Заказчика; иногда от Обихода как технический заказчик)
3. Выезд альпиниста (3-я группа по высоте) + бригадира + Газели с прицепом
4. Спил по частям (для дерева Ø свыше 30 см или с риском падения) или обычный спил у основания
5. Корчевание пня (если в наряде) — корчеватель до Ø 60 см или фреза до уровня грунта
6. Щепа из ветвей (барабанный измельчитель) или вывоз порубочных остатков (Газель прицеп — ФККО 7 31 200 01 72 4)
7. Акт + фото-отчёт «до/после»

### 2.3 · chistka-krysh

| Поле | Значение |
|---|---|
| **ОКВЭД** | 81.21 (общая уборка зданий) + 81.29 (другая чистка) |
| **ГОСТ** | ГОСТ Р 50571.5-94 (электробезопасность зданий — для работ на крышах с электропроводкой) |
| **СНиП / СП** | СП 17.13330.2017 (Кровли — нормативные требования к содержанию) · СНиП 12-04-2002 → Минтруда №782н |
| **ЖК / постановления** | ЖК РФ ст. 162 (содержание общего имущества МКД — обязанность УК очищать кровли) · Постановление Госстроя 170 (правила технической эксплуатации жилищного фонда) |
| **КоАП** | ст. 7.22 (нарушение правил содержания жилого помещения — штраф УК до 50 тыс. ₽; в случае ЧП от наледи — до 250 тыс. ₽ + ущерб) |
| **Допуски** | Минтруда №782н 3-я группа (для МКД крыш свыше 5м) или 2-я группа (для частных домов до 5м) |
| **СРО** | СРО строителей если работы по ремонту кровли (не обычная очистка); обычная очистка — не требуется |

**Тех. процесс:**

1. Заявка от УК / ТСЖ / частника → фото крыши + сегмент (МКД / частный) + аварийность (наледь нависает, риск ЧП)
2. Выезд бригады + альпиниста (3-я группа) или оператора автовышки 22 м
3. Проверка погоды (отрицательная температура для безопасности альпиниста — sustained Минтруда №782н)
4. Очистка снега и наледи **строго ручным инструментом** (лопата, скребок) — не лом + не ударные инструменты (запрещено по СНиП — повреждение кровельного покрытия)
5. Сброс снега во двор → погрузка в Газель / Камаз (если требуется вывоз; ФККО 8 19 100 01 51 5)
6. Фото-отчёт «до/после» + акт по форме УК

| **ГОСТ** | ГОСТ Р 53231-2008 (бетонные конструкции — методы испытаний при разборке) · ГОСТ 31937-2024 (мониторинг технического состояния) |

| **КоАП** | ст. 9.5 (нарушение градостроительной деятельности — штраф юр.лицу 500 тыс. - 1 млн ₽ при сносе без разрешения капитальной постройки) |
| **Допуски** | Минтруда №782н группа 2-3 (если работы на высоте при демонтаже кровель) |
| **СРО** | **СРО строителей обязательна** для демонтажа капитальных строений + работ в коттеджных посёлках (Градкодекс ст. 55.5) — sustained `credentials.md` §2 |

**Тех. процесс:**

1. Заявка → фото объекта + классификация (капитальная / некапитальная — fundament + размер) + год постройки + материалы
2. Если капитальная (с фундаментом, площадь > 30 м²) → проверка разрешения на снос (от УК / Заказчика) — иначе ст. 9.5 КоАП риск
3. Выезд бригады + бригадир + гидравлические ножницы / болгарка по бетону / кувалды
4. Демонтаж по технологии: **сверху вниз** (sustained СП 325.1325800) — кровля → стены → фундамент
5. Сортировка отходов по ФККО: 7 12 200 01 22 4 (грунт+бетон+кирпич) + 4 05 122 11 60 4 (древесина обработанная)
6. Транспортировка на лицензированный полигон (Газель / Камаз)
7. Расчистка территории + акт + фото-отчёт «до / процесс / после»
