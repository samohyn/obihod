# chat/ — логи сессий Claude Code

Каждая сессия с Claude Code = один markdown-файл `YYYY-MM-DD-<topic>.md`. Папка **в git НЕ трекается** (см. железное правило #7 в [CLAUDE.md](../CLAUDE.md)) — существует только локально.

## Зачем нужно

Дополняет `.claude/memory/handoff.md` (короткий statusline на 30 секунд) **полным контекстом сессии** — обсуждения, цитаты оператора, развилки, обоснования решений. Нужен для:

- восстановления контекста через 2-3 недели, когда handoff уже устарел
- references при retrospective («почему мы это решили?»)
- архива обсуждений, которые не перешли в формальные артефакты `specs/<EPIC>/`

## Формат файла

- **Имя:** `YYYY-MM-DD-<topic>.md`. Дата = старт сессии. Тема — kebab-case, 2-4 слова. Примеры: `2026-04-29-seo-discovery-kickoff.md`, `2026-04-29-panel-backlog-rebuild.md`.
- **Содержимое (рекомендация, не строгая):**
  - Header: дата, длительность, основная роль / тема
  - Summary — 5-10 строк
  - Ключевые цитаты оператора (raw, дословно)
  - Final decisions — что принято, кем, почему
  - Созданные / изменённые артефакты (со ссылками на пути в репо)
  - Open follow-ups (что осталось)

## Когда писать

В конце сессии (перед `/clear` или закрытием) или при существенной развилке посреди сессии. Параллельно обновляется `.claude/memory/handoff.md` (короткий срез) и, если применимо, `.claude/memory/learnings.md` (правила).

## Что НЕ сохранять

- Секреты (`.env`, токены, пароли, ключи)
- Полные транскрипты — только curated summary с цитатами по существу
- Дублирующее содержимое из `specs/<EPIC>/intake.md` или `.claude/memory/handoff.md` — ссылайся, не копируй

## Связанные источники

- [`.claude/memory/handoff.md`](../.claude/memory/handoff.md) — короткий статус для следующей сессии (трекается в git)
- [`.claude/memory/learnings.md`](../.claude/memory/learnings.md) — уроки + правила (трекается в git)
- [`specs/<EPIC>/intake.md`](../specs/) — формальные spec-артефакты (трекаются в git)
- [`team/cpo-syncs/YYYY-MM-DD.md`](../team/) — cross-team синки (трекаются в git)

`chat/` — единственный из этих источников, который **локальный и не светится в репо**.
