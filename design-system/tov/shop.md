## 1. Архетип и роль продавца

**Caregiver (50%) + Sage (50%).** Ruler-нота из основного бренда уходит — в магазине мы не «бригадир, который закрыл штраф ГЖИ», а **питомниковод, который знает свой материал**.

| Что значит на практике |
|---|
| Говорим о растении как об **объекте знаний**, не как о товаре. Рассказываем сорт, подвой, зону зимостойкости, габитус, опылитель — потому что это влияет на приживаемость, а не для маркетинговой важности. |
| Покупатель — **профи-садовод или опытный дачник**. Новичка не отталкиваем, но язык не упрощаем «для чайников». Если читатель не знает термин — даём короткое пояснение в скобках, а не переходим на бытовой уровень. |
| **B2B (опт, ландшафтники)** — не отдельный голос, а UI-чекбокс «оплата юр. лицом» в чекауте. Опт-сетки в карточках по умолчанию убраны. |
| Caregiver проявляется в **гарантии приживаемости**, упаковке ЗКС/ОКС, инструкциях по посадке. Sage — в редакторских сравнениях сортов, ботанических справках, рекомендациях по подвою и зоне. |

## 2. Шкалы голоса

```
Funny        ◄────────────●───────► Serious        (серьёзный, спокойный)
Formal       ◄─────●─────────────► Casual         (профи-разговорный, без канцелярита)
Respectful   ◄●──────────────────► Irreverent     (уважение к читателю и материалу)
Enthusiastic ◄────────────●──────► Matter-of-fact (по делу, без восторгов)
Authority    ◄────●──────────────► Equal          (как с равным, не сверху)
```

«Авторитет vs равный» — отдельная shop-шкала: говорим **как с равным**, не свысока. Питомниковод не учит — он сравнивает сорта вслух, а покупатель сам выбирает.

## 3. Голосовые принципы

1. **«Сорт сам себя продаёт».** Описание = ботанические факты + поведение в саду + честные ограничения. Без «эксклюзивный», «лучший», «суперсорт».
2. **«Приживётся или вернём».** Гарантия в цифрах и условиях, не в слогане. «Замена за наш счёт в первый сезон при соблюдении инструкции по посадке.»
3. **«Сезон — это срок, не повод спешить».** Сообщаем окно посадки (апрель–май, сентябрь–октябрь), не давим «успей».
4. **«Каталог — не магазин, а справочник».** Карточка должна быть полезна **даже если посетитель не купит сегодня** — он вернётся за информацией в следующий сезон.
5. **«Контейнер и подвой — обязательная часть имени товара».** Не «Яблоня Антоновка», а «Яблоня Антоновка обыкновенная, подвой 54-118, C7,5».

## 4. Доменная лексика — Do / Don't

### Используем как есть (и поясняем в скобках при первом упоминании в карточке)

| Термин | Короткое пояснение для UI helper |
|---|---|
| **ЗКС** | закрытая корневая система — растение в контейнере, можно сажать с весны до осени |
| **ОКС** | открытая корневая система — без контейнера, копаем перед отправкой, сажаем только в межсезонье |
| **Подвой** | корневая основа привитого саженца; определяет высоту и зимостойкость |
| **Подвой 54-118** | полукарликовый яблоневый подвой, дерево 3,5–4 м, плодоношение с 3–4 года |
| **Подвой М9** | карликовый, дерево 2–2,5 м, требует опоры |
| **Зона зимостойкости** | USDA-зона; для МО рабочие зоны 4–5, всё ниже — риск |
| **Габитус** | форма кроны взрослого растения (пирамидальный, шаровидный, плакучий) |
| **Опылитель** | сорт-сосед для перекрёстного опыления; без него урожая может не быть |
| **Контейнер C7,5 / C10 / C25** | объём горшка в литрах; влияет на возраст саженца и приживаемость |
| **Привитой / корнесобственный** | способ размножения; влияет на повторяемость сортовых признаков |
| **Двухлетка / трёхлетка** | возраст саженца; коррелирует с ценой и сроком до плодоношения |
| **Крупномер** | дерево от 2 м с комом земли; отдельная логистика и посадка |

### Anti-words shop (запрет, проверка хуком — см. §10)

| Don't | Почему |
|---|---|
| «магазинчик» | Уменьшительное → подрывает Sage-авторитет. Мы — питомник. |
| «эксклюзивный» | Маркетинговая вода. У саженца есть сорт и подвой, не «эксклюзивность». |
| «суперсорт», «лучший сорт», «топ-сорт» | Сорт оценивается по характеристикам, не маркетинговым превосходным степеням. |
| «спешите», «успейте», «не упустите» | Давление времени. У нас сезон — окно, не дедлайн. |
| «осталось N шт» (как FOMO-маркер) | Реальный остаток показываем как факт стока, не как pressure-мотив. Запрет — на формулировку, не на цифру. |
| «органический» (в маркетинговом смысле) | Если есть сертификация — конкретно её называем. Иначе — не используем. |
| «настоящий», «правильный» (как эпитеты сорта) | «Настоящая Антоновка» — нет такой характеристики. Есть подвой, происхождение, поставщик. |
| «уникальный», «премиум», «элитный» | Те же причины — пустые эпитеты. |
| «хит продаж», «бестселлер» | Социальное доказательство ставим цифрой («продано 240 шт за сезон 2025»), не лейблом. |

> Анти-words services из основного бренда (`услуги населению`, `имеем честь`, `от 1 000 ₽`, `в кратчайшие сроки`, `индивидуальный подход`) — продолжают действовать и в shop.

## 5. Глаголы и формулы

### Действие магазина (мы)

«приживётся» / «выкопаем с комом» / «довезём» / «упакуем» / «подберём сорт под зону» / «подскажем опылитель» / «зарезервируем на сезон» / «закрепим позицию» / «заменим по гарантии».

### Действие покупателя (вы)

«посадите» / «сажайте» / «подбирайте» / «учитывайте зону» / «оставьте опору» (для М9) — **только в инструкциях по посадке**, не в продающем тексте.

### Сезонные

«окно посадки апрель–май», «сентябрь–октябрь», «ОКС только до распускания почек», «зарезервируем на весну 2027», «закрепим позицию на сезон».

## 6. Пять голосовых сценариев

### 6.1 Карточка товара (description)

> **Яблоня Антоновка обыкновенная, подвой 54-118, C7,5**
>
> Зимний сорт народной селекции, проверенный на МО более ста лет. На полукарликовом 54-118 дерево вырастает до 3,5–4 м, в плодоношение входит на 3–4 год. Зимостойкость — зона 4, переносит –40 °C без укрытия.
>
> Опылители — Осеннее полосатое, Уэлси, Мелба. Без соседа-опылителя завязь идёт слабо.
>
> Контейнер C7,5, корневая закрытая — сажайте с апреля по октябрь. ОКС-вариант доступен в марте и октябре.

### 6.2 Фильтр / категория

> **Яблони на полукарликовом подвое (54-118)**
>
> 18 сортов · от 1 200 ₽ · окно посадки апрель–октябрь (ЗКС)
>
> Полукарлик подходит для участков 6–10 соток. Дерево держится 25–30 лет, не требует опоры в отличие от карликового М9. Под МО — рабочая зимостойкость от зоны 4.

### 6.3 Чекаут с B2B-опцией

> Доставим в Раменское 2026-05-04, до 14:00. Бесплатно при заказе от 8 000 ₽.
>
> ☐ Оплата от юр. лица (счёт + закрывающие документы)
>
> Срок резерва — 3 дня после оплаты. Если посадка позже — закрепим позицию на сезон без доплаты.

### 6.4 Бронь на сезон (out of stock в межсезонье)

> Сорт выкопан, новая партия — апрель 2027.
>
> Зарезервируем по предзаказу: вы фиксируете цену сезона 2026, мы откладываем саженец, как только партия поступит. Предоплата 30%, остаток — при отгрузке.
>
> [Зарезервировать на весну 2027]

### 6.5 После заявки (toast / email)

> Заявка ОБИ-2026-S-00118 принята.
>
> Менеджер питомника свяжется в течение рабочего дня, подтвердит наличие подвоя 54-118 в нужном размере и сроки доставки. Если что-то по сорту лучше заменить — предложим альтернативу с сопоставимой зимостойкостью и сроком плодоношения.

## 7. Типографика и числа

Наследуем из основного microcopy (`design-system/foundations/microcopy.md`):
- Рубли: `1 200 ₽` (nbsp между тысячами, пробел перед ₽)
- Диапазон цен / размеров: длинное тире — `1 200 – 4 800 ₽`, `3,5–4 м`
- Десятичные размеры: запятая — `3,5 м`, `C7,5`
- Размер контейнера: латиница `C` + цифра — `C7,5`, `C10`, `C25`
- Подвой: `подвой 54-118` (через дефис, без пробелов)
- Зона зимостойкости: `зона 4`, `зоны 4–5`
- Возраст саженца: `двухлетка`, `трёхлетка` (одним словом)
- Сезон: `апрель–май` (длинное тире, без пробелов внутри пары месяцев)

## 9. Photography mood (placeholder)

Источник базовых правил — [foundations/photography.md](../foundations/photography.md). Дельта для shop:

- **Растение в питомнике** — снимаем с комом, в контейнере, не «на белом фоне». Видно подвой, корневую шейку, прививку.
- **Карточка сорта** — три кадра: общий вид саженца, плод/цветок крупно (если сезон), взрослое дерево из питомника-родителя (или ботанический архив с подписью).
- **Размер в кадре** — рядом ставим складной метр или человека для масштаба, особенно для крупномеров.
- **Сезонность видна** — апрельский саженец до распускания и тот же сорт в плодоношении должны быть в одной карточке (если есть архив).
- **Упаковка** — ОКС с увлажнённой мешковиной и стяжкой, ЗКС в маркированном контейнере с этикеткой сорта. Это часть бренда, не «техническое фото».

**Wave 1 backlog (расширения):**
- Фото-протокол для крупномеров (большой контейнер, кран, посадочная яма)
- Cross-sell-фото опылителя (рядом с основным сортом, парный кадр)

## 10. Hook anti-words (handoff `do` / `tamd`)

После апрува v1.0 в [.claude/hooks/protect-immutable.sh](../../.claude/hooks/protect-immutable.sh) добавляется shop-набор anti-words с областью действия — пути `apps/shop/`.

```bash
# pseudo, реальная имплементация — в protect-immutable.sh
SHOP_ANTI_WORDS=(
  "магазинчик"
  "эксклюзивный"
  "суперсорт" "лучший сорт" "топ-сорт"
  "спешите" "успейте" "не упустите"
  "органический"   # маркетинговый, не сертификационный
  "настоящий" "правильный"   # как эпитеты сорта
  "уникальный" "премиум" "элитный"
  "хит продаж" "бестселлер"
)
# Whitelist через escape-маркер `obihod:ok` в той же строке (для цитат и контр-примеров).
```

«Осталось N шт» — хук **не блокирует** числовое значение остатка; блокирует только формулировки типа «осталось всего», «торопитесь — осталось», «последние».

## 11. Visual heritage

Visual язык shop **полностью наследуется** из основного [brand-guide.html](../brand-guide.html):
- Палитра, типографика (Golos Text + JetBrains Mono), радиусы, spacing, shadow, motion.
- Логотип `ОБИХОД` остаётся базовым; для shop добавляется suffix `· магазин` в header в стиле `gh-wordmark` основного гайда.
- Иконография services не переиспользуется в shop без адаптации — питомник просит свой набор (контейнер, прививка, подвой, опылитель, зона). Это backlog Wave 2.

В рендере для команды (`brand-guide-shop.html`) визуальные секции — короткая reference-полоса со ссылкой на основной guide; основной объём — **TOV-shop, лексика, компоненты магазина, photography delta**.
