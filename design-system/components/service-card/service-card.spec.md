# ServiceCard · spec

**Статус:** v0.9 · draft · роль `ui` · утверждено `art`
**Реализация:** [site/components/marketing/sections/Services.tsx](../../../site/components/marketing/sections/Services.tsx)
**Бизнес-контекст:** карточка одной из 4 услуг (арбористика / крыши / мусор / демонтаж). Живёт в сетке на главной и в каталоге услуг. Цель — за 3 сек донести «что делаем» и перевести на LP услуги.

## Anatomy

```
┌────────────────────────────────────┐
│ [icon/graphic 48×48]              │  ← glyph из brand (хвоя/снежинка/кирпич/навес)
│                                    │
│ Арбористика                       │  ← h-m
│                                    │
│ Спил, кронирование, удаление      │  ← body-sm muted
│ пней. Фикс-цена за объект.        │
│                                    │
│ от ₽12 800 / объект               │  ← mono.tnum primary — якорь цены
│                                    │
│ [ Подробнее →  ]                  │  ← btn-ghost, полная карточка кликабельна
└────────────────────────────────────┘
```

## Variants

| Variant | Когда |
|---|---|
| `default` | Сетка «Наши услуги» на главной (4 карточки) |
| `district` | Сетка на programmatic LP района (с локализованным подзаголовком: «в Раменском») |
| `compact` | Inline-связка в блоге / FAQ — без цены, только название + glyph |

## Services × visual glyph

| Кластер | Glyph из Круга сезонов | Где взять |
|---|---|---|
| Арбористика | NW — ёлочка herringbone | `BrandLogo.mark.svg` NW-quadrant |
| Крыши | NE — снежинка роза ветров | NE-quadrant |
| Мусор / вывоз | SE — навес со стрелками | SE-quadrant |
| Демонтаж | SW — план-грид двора | SW-quadrant |

Glyph вынесен из знака — **подтверждает** связь всех услуг с брендом. Цвет `currentColor`, родитель задаёт `color: var(--c-primary)`.

## States

| State | Border | Background | Shadow | Lift |
|---|---|---|---|---|
| `default` | 1px `var(--c-line)` | `var(--c-card)` | none | — |
| `hover` | 1px `var(--c-primary)` | `var(--c-card)` | `{shadow.sm}` | `translateY(-2px)` |
| `focus-within` | 2px `var(--c-primary)` | `var(--c-card)` | `{shadow.focus-ring}` | — |
| `active` | 1px `var(--c-primary)` | `var(--c-bg-alt)` | none | 0 |

**Вся карточка — ссылка** (`<a>`), внутренняя btn-ghost декоративно-дублирующая, `tabIndex={-1}` чтобы не двойной tab-stop.

## Copy

### Название (h-m)
- Арбористика
- Чистка крыш
- Вывоз мусора
- Демонтаж

(Короткое, без «услуги» / «работы» суффиксов.)

### Подзаголовок (body-sm muted)
Одно предложение, перечисление услуг + якорь фикс-цены.

- «Спил, кронирование, удаление пней. Фикс-цена за объект.»
- «Снег и наледь с крыш. Работаем на высоте, страхуем.»
- «Бытовой и строительный. Контейнер — 7 / 20 / 30 м³.»
- «Сараи, перегородки, старые строения. Вывозим мусор.»

### Цена якорь (mono.tnum)

Обязательна. Формат `от 12 800 ₽ / объект` — но это не «от», это **стартовая тариф-линия**, фикс-цена разная у разных объёмов.

«от» оставляем только когда реально работает амплитуда (спил может быть 12 800 ₽ и 85 000 ₽). Если услуга фикс-табличная → `12 800 ₽ / объект` без «от».

## Token usage

- Padding: `{spacing.6}` = 24 px по всем сторонам
- Border-radius: `{radius.xl}` = 16 px
- Transition: `{motion.presets.hover}`
- Gap между элементами внутри карточки: 12-16 px

## A11y

- **Целиком link:** `<a href="/uslugi/{slug}" class="service-card">` — структура heading (h3), paragraph, цена — всё внутри.
- **Image / icon:** декоративный SVG → `aria-hidden="true"`.
- **Focus:** focus-within style с ring.
- **Touch-target:** целая карточка ≥ 200×180 — выше WCAG min.
- **Hover-only cues запрещены** — translate/shadow это бонус, не единственный signal «это кликабельно». Курсор pointer обязателен.

## Responsive

| Breakpoint | Grid | Card |
|---|---|---|
| ≤ 640 | 1 col | full-width stack |
| 641-1023 | 2 col | half-width |
| ≥ 1024 | 4 col | quarter-width |

На мобиле карточки стоят **полной ширины** (не 2 узких) — чтобы цена и описание оставались читаемыми.

## Edge cases

- **Длинное название услуги:** max 2 строки, balance. `line-clamp: 2` если нужно.
- **Нет цены (услуга «по звонку»):** якорь заменяется на «Цена по объёму» body-sm italic — не скрываем, чтобы не терялся ритм сетки.
- **Иконка нет:** fallback — `LogoMark.mark-simple` (круг с О). Не делаем пустую карточку.
